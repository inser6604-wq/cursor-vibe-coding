# PRD (Product Requirements Document)
## AETHER — Space Travel Landing Page

# 1. 프로젝트 개요

## 프로젝트명
AETHER

## 프로젝트 유형
프리미엄 우주여행 브랜드 랜딩페이지

## 프로젝트 목적
AETHER는 미래 우주여행 브랜드를 컨셉으로 한 인터랙티브 랜딩페이지 프로젝트이다.  
사용자에게 단순한 정보 전달이 아닌, 우주를 “경험하는 듯한 몰입감”을 제공하는 것을 목표로 한다.

시네마틱한 비주얼과 스크롤 기반 인터랙션을 활용하여 브랜드의 미래적이고 럭셔리한 이미지를 강조한다.

# 2. 프로젝트 목표

- 브랜드 아이덴티티를 시각적으로 강하게 전달
- 몰입감 있는 랜딩페이지 UX 구현
- 스크롤 기반 인터랙션 경험 제공
- 반응형 웹 구현
- 포트폴리오용 고퀄리티 UI 제작
- 미래지향적 브랜드 무드 표현
